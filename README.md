# MultipleDisturbances
Scripts in this folder bring in data from the Yosemite (YOSE) project Transcribed Data folder on Google Drive, clean the data, and run analyses related to the multiple disturbances paper. 

The occurence and event scripts are data carpentry scripts designed to transform the raw data from the Google Drive into a GBIF compatible format that can be published with other NPS data products. The occurence script contains tree-level data and corresponds to the occurence tab of the NPS data template. The event script contains plot-level data and corresponds to the event tab of the NPS data template. 

Scripts with numbers in the name are designed to be run in order for a given analysis, but can also be used as building blocks for a different work flow. For the understory script series, script 0 brings in a user defined function into the global environment. Script 1 reads in the raw data from the Google Drive and applies the function to transform the data from a series of lists to a data frame and converts the data format from wide to long format. Script 2 cleans up the raw data by recoding unknown species, and taking care of misspellings and other inconsistencies. Script 3 calculates species richness for understory species. Script 4 conducts a preliminary EDA with the understory richness data.
